# MyTrader
### 설치파일
* 키움 Open API+
* 키움 번개3
* 키움 KOA Studio

### 코드 설명
1. Kiwoom.py
    Kiwoom 서버 접속, 데이터 요청 및 이벤트 처리

2. MyTrader.py
    Kiwoom Class를 이용하여 Trader Window 및 기능 구현
    기능 설명 추후...

3. Kakao.py
    카톡 나에게 메시지 보내기 기능 담당

4. Autorun.py
    번개 자동 실행

4. chromedriver
    selenium 사용을 위해 설치 필요

5. kakaoInfo.txt
    Kakao ID
    Kakao PW

6. bungaeInfo.txt
    Kiwoom ID
    Kiwoom 공인인증서 PW
